# Quiero_regresar_a_casa
## Technology Stack:
### BackEnd
1. Python 3.11
2. Django 4.1.2 (https://docs.djangoproject.com/en/4.1/)
   - Django rest framework (https://www.django-rest-framework.org/)
   - Django-phonenumber (https://django-phonenumber-field.readthedocs.io/en/latest/)

### FrontEnd
1. React.js
2. Less (https://lesscss.org/)
3. Tailwind (https://tailwindui.com/documentation)

## Run instruction
To run the backend:
```
docker-compose -f backend/docker-compose.yaml up
```
